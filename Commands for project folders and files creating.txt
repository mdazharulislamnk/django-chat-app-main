Here are Windows CMD commands to create just the empty folders and placeholder files that match your project’s layout; the SQLite database file will be created automatically by Django after migrations.​
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------


Create folders
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
mkdir django-chat-app-main && cd django-chat-app-main​

mkdir djangochat chat templates​
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------


Create project package files
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
type NUL > manage.py​

cd djangochat && (type NUL > init.py) && (type NUL > settings.py) && (type NUL > urls.py) && (type NUL > asgi.py) && (type NUL > wsgi.py) && cd ..​
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------


Create app files
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
cd chat && (type NUL > init.py) && (type NUL > apps.py) && (type NUL > admin.py) && (type NUL > models.py) && (type NUL > views.py) && (type NUL > urls.py) && (type NUL > tests.py) && cd ..​
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------


Create templates
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
cd templates && (type NUL > home.html) && (type NUL > room.html) && cd ..​
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------


Notes:
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Use init.py (with double underscores) so Python recognizes the djangochat and chat directories as packages; init.py without underscores will not work.​

Do not create db.sqlite3 manually; it is generated by Django after python manage.py migrate based on DATABASES in settings.py.
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
