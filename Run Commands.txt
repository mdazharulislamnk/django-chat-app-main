Install Django
----------------------------------------------------------------------------------------------------------
pip install django
----------------------------------------------------------------------------------------------------------

Initialize database
----------------------------------------------------------------------------------------------------------
python manage.py makemigrations chat

python manage.py migrate
----------------------------------------------------------------------------------------------------------

Run the server
----------------------------------------------------------------------------------------------------------
python manage.py runserver 127.0.0.1:8080 (use another free port if needed)
----------------------------------------------------------------------------------------------------------

Create a secret key
----------------------------------------------------------------------------------------------------------
python -c "from django.core.management.utils import get_random_secret_key as g; print(g())"

Then set in djangochat/settings.py:

SECRET_KEY = 'paste-generated-key'
----------------------------------------------------------------------------------------------------------
